Package Health Extension Settings

Configure the API URL for your local Package Health API server

Enter the URL where your Package Health API is running. Make sure to include the protocol (http:// or https://) and port number if needed.

Common Examples:

  • http://localhost:8000
  • http://localhost:3001
  • http://127.0.0.1:8000
  • https://your-api-domain.com

Troubleshooting:

  • • Make sure your API server is running
  • • Check that the port number is correct
  • • Verify the URL includes http:// or https://
  • • Try 127.0.0.1 instead of localhost if having issues
  • • Check browser console for detailed error messages
